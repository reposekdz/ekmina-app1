## 📊 CALCULATIONS & ALGORITHMS

### Investment Calculations ✅
```dart
✅ Interest Rate = Base Rate + Duration Bonus + Tier Bonus
✅ Expected Return = Amount × (1 + Interest Rate × Days/365)
✅ Profit = Expected Return - Amount
✅ Daily Return = Profit / Days
✅ Per Second Growth = Daily Return / 86400
✅ Current Value = Amount + (Per Second × Seconds Elapsed)
```

### Dividend Calculations ✅
```dart
✅ Member Share = Member Shares / Total Shares
✅ Member Dividend = Total Profit × Member Share
✅ Distribution = Based on cycle (weekly/monthly)
```

### Group Financial Calculations ✅
```dart
✅ Escrow Balance = Deposits - Loans - Withdrawals
✅ Available Capital = Escrow - Reserved Funds
✅ Interest Earned = Σ(Loan Payments - Principal)
✅ Penalties = Σ(Late Payments × Penalty Rate)
```
